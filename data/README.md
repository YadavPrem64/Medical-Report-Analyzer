# Data Directory

This directory contains all data used in the Medical Report Analyzer project.

## Structure

- `raw/` - Original medical reports (not tracked in Git for privacy)
- `processed/` - Cleaned and processed data
- `samples/` - Sample reports for testing

## Datasets

### MIMIC-III
- Download from: https://physionet.org/content/mimiciii/1.4/
- Requires registration and training

### i2b2 NLP
- Download from: https://www.i2b2.org/NLP/DataSets/

## Privacy Notice

⚠️ **IMPORTANT**: Never commit real patient data to Git! 
All personal health information should remain local and encrypted.
